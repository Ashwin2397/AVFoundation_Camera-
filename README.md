#Overview


This is a prototype app to let our clients have a feel of voice assisted camera control 

## Iterations 

### Number 1
* Allows for capturing the image 
* Viewing image on another view
* Ability to save image on another view 
* No Zoom functionality

### Number 2
* No display of picture 
* Picture gets auto saved in to the photo library 
* A flash animation has been added to indicate that the picture has been taken 
* Zoom test has been implemented
* The other view for displaying the picture is still there and has not been deleted
	* Can be switched off via a function call  
### Number 3
* Similar to number 2 
	* Except the fluff has been removed
	* More comments for my own sake has been added as well 
* It has a uitextfield to grab the zoom value and send it to the function 

### Number 4
* Orientation functionality added 
* All else equal to 'Number 3'

### Number 5
* UItextfield has been removed
* All else equal
