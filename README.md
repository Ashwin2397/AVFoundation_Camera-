#Overview

This is a prototype app to let our clients have a feel of voice assisted camera control 

## Iterations 

### Number 1
* Allows for capturing the image 
* Viewing image on another view
* Ability to save image on another view 
* No Zoom functionality

### Number 2
* No display of picture 
* Picture gets auto saved in to the photo library 
* A flash animation has been added to indicate that the picture has been taken 
* Zoom test has been implemented
